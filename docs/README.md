# Documentation

Documentation for the **Project Landscape Mapper** — a client-only Vue 3 SPA that maps
project milestones across swimlanes and phases.

- [getting-started.md](./getting-started.md) — how to install and run the app, and where data is stored.
- [milestone-data.md](./milestone-data.md) — the data a milestone holds (the `L3Milestone` model and its `connections`).
- [modelling-pane.md](./modelling-pane.md) — the central editor canvas: nodes, layout, interactions, and the hover-to-reveal connection lines.
- [admin-settings.md](./admin-settings.md) — the Admin view: streams, phases, gates, and roles.
- [standard-template-data.md](./standard-template-data.md) — the "Load sample" template (phases, gates, streams, roles, 17 milestones, seeded connections).
